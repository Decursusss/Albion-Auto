# AlbionAuto
Автоматизация для Альбиона при помощи python

# Инструкция

1) Клонируем репризиторий
2) Запускаем команду pip install -r requirements.txt
3) Запускаем саму игру в оконом режиме (1440x900)
4) Запускаем программу от имени администратора через python main_gui.py или же в EDI просто указываете чтобы программа запускалась через main_gui.py
5) Для работы скрипта найдите область рядом с водой и просто нажмите в интерфейсе Start (и кликните на окно игры / так как есть нюансы связанные с античитом фокусировка на экране была отключена поэтому после запуска просто кликните на экран игры)
6) Для отключения нажимаем Stop
7) Чтобы просматривать логи определеннго сервиса выберить его во вкладке и внизу списка будет калочка Debug
8) Если вы хотите подключить Телеграм уведомления то создайте бота через BotFather и вставьте свой BotToken + укажите ChatId

# Дополнительная информация
* Вы можете посмотреть принцип работы пройдя по ссылкам ниже на Youtube. Так же советую для точности работы скопировать интерфейс как это у меня сделано в видео.
* Так как тестировал программу на Т4 локаций нейросеть только может понимать деревья от Т2 до Т4 если хотите чтобы были и другие тиры то вы должны будете доубучить ее. Сам датасет составлял примерно 1000 изоображений с аугментацией.
* Так же во время тестов понял что лучше всего использывать ее просто для рыбалки так получаеться больший доход в час.
* Есть нюанс с водным кластером там указывалась темная вода в маске поэтому если вода будет светлой то по хорошему просто поменяйте список Хекс значений на светлое для водного кластера
